## **Vira Global Chat Bot** 

## **NODE.JS V13 COMPATIBLE** 

Welcome to the official deployment and configuration guide for the **Vira Global Chat Bot** . This Discord bot bridges communication between multiple servers, enabling cross-guild channels to interact seamlessly in real time. 

## **1. Quick Start & Installation** 

Setting up the Vira Global Chat Bot is simple and straightforward. Follow the steps below to configure your instance: 

1. Download the configuration template file: **config.json** from the official repository. 

2. Open the file in any text editor and insert your unique Discord Bot Token into the designated field. 

```
// Repository Link for Configuration Template https://github.com/Tira-tw/Vira-
global-chat-bot/blob/main/config.json
```

## **2. Cross-Server Chat Setup (Global Chat)** 

To establish a multi-server communication link and bind a specific channel to the global network, execute the following command in your server: 

```
[prefix]global <#channel>
```

_Example usage:_ `!global #global-chat` . Once configured, messages sent within this designated channel will automatically broadcast to all other participating servers connected via the bot.
